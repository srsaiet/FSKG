# Flight Simulator Deluxe KeyGen



Solo basta elegir el número de claves a generar y presionar el botón generar ….



Notas:  



&#x09;\* El archivo original pkeyconfig.xrm-ms fue renombrado como pkconfig.xmr-ms

&#x09;\* El generador usa el algoritmo msft:rm/algorithm/pkey/2005 mismo que esta en el archivo pkconfig.xrm-ms

&#x09;\* La versión del algoritmo no requiere ninguna firma RSA 2048 (como Windows / Office)
